Apakah aman menyimpan dana di OneWallet?

Sangat aman. OneWallet menggunakan enkripsi standar militer AES-256 dan protokol BIP39 untuk frase pemulihan. Selama Anda menjaga frase Mnemonic tetap rahasia, dana Anda aman.

Apa itu Jaringan ONE?

Jaringan ONE (OneChain) adalah blockchain Layer-1 berkecepatan tinggi yang dirancang untuk skalabilitas massal dengan biaya transaksi yang mendekati nol.

// OneChain RPC Endpoint
const RPC_URL = "https://regardlessly-foundationary-tawanda.ngrok-free.dev";

// Mengambil saldo terbaru dari dompet aktif
async function refreshBalance() {
    try {
        const res = await fetch(`${RPC_URL}/balance/${myWallet.address}`);
        const data = await res.json();
        
        // Update UI dengan saldo (format: 0.00 ONE)
        console.log(`Balance: ${data.balance} ONE`);
        return data.balance;
    } catch (err) {
        console.error("Gagal terhubung ke RPC Node");
    }
}
